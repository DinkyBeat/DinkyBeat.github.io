About

Olave processes historical NFL play-by-play datasets using nflfastR to compute a set of advanced metrics beyond box-score stats. The goal is a reproducible research framework that can eventually support real-time streaming during live games via Apache Kafka or Redis Streams.

The notebook metrics1.ipynb is the main entry point — run it interactively via Binder (no setup required) or clone the repo locally.

Python R / nflfastR Pandas Plotly Jupyter PostgreSQL Kafka (planned)
