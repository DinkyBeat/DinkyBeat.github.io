NFL ANALYTICS
Advanced metrics framework · Python · R · Jupyter
IN PROGRESS
GitHub ↗

ABOUT

A research-oriented framework for computing advanced NFL metrics from play-by-play data. Currently processes historical datasets using nflfastR, with future plans for real-time streaming during live games via Apache Kafka or Redis Streams.

METRICS COMPUTED

EPA
Expected points added per play
SUCCESS RATE
Play-level success frequency
WPA
Win probability added
RED ZONE
Red zone efficiency by team
DRIVE EFF.
Drive-level scoring efficiency
COMPARATIVE
Cross-season team rankings

STACK

Python R / nflfastR Pandas Plotly Jupyter PostgreSQL Kafka (planned)
