Olave — NFL Analytics

Advanced metrics framework · Python · R · Jupyter

In Progress GitHub ↗

A research-oriented framework for computing advanced NFL metrics from play-by-play data. Processes historical datasets using nflfastR, with future plans for real-time streaming during live games. Hosted live at olave.govardha.net ↗.
