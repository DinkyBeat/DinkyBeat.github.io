Use a linen off-white minimalist aesthetic. The reference is headlandstech.com: clean, no animations or gimmicks, generous whitespace, simple nav.

**Why:** User explicitly scrapped the previous dark neo-expressionist/3D gimmick approach and replaced it with a light minimal design.

**How to apply:** Never revert to dark backgrounds, grid overlays, scanlines, flicker animations, 3D canvases, or other "gimmicks". Keep it clean and restrained. Current palette: `--bg: #F6F3ED`, `--text: #1A1815`, `--muted: #7A7468`, `--border: #D8D3C8`. Fonts: Headland One for headings/name, Inter for body.
