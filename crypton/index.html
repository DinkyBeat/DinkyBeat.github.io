How to play

Definition
A synonym or direct description of the answer, usually at the start or end of the clue.
Indicator
A signal word that tells you how to manipulate the fodder — e.g. confused, reversed, around.
Fodder
The letters or word(s) you actually work with to derive the answer.
  • Both players enter the same room and see the same clue simultaneously.
  • First player to submit the correct answer wins the point for that clue.
  • Each game is 5 clues. You have 90 seconds per clue.
  • After each clue, the answer and breakdown are revealed.
  • Highest score at the end wins.
